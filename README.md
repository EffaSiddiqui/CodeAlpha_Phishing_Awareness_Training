# Phishing Awareness Training Module

## Project Overview
This project is a comprehensive training module designed to educate users on how to identify and prevent phishing attacks. Created during my Cyber Security internship at CodeAlpha.

## Content Covered
- What is Phishing and Social Engineering?
- Common Red Flags in Emails and Websites.
- Real-world examples of Phishing attacks.
- Best practices for staying safe online.

- **Format:** PDF Presentation
- **Tool Used:** Canva / Graphic Design
